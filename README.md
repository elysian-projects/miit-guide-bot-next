# MIIT Guide Bot

Telegram бот, который проводит экскурсию по территории университета РУТ (МИИТ).

## Цели проекта

- познакомить студентов и гостей ВУЗа с историей РУТ (МИИТ)

## Запуск

- установить `docker` на компьютер
- создать в корне файл `.env` и скопировать в него:

  ```
  # Telegram
  TOKEN=<your_token>

  # Database
  HOST="127.0.0.1"
  DBNAME="miit-guide-bot"
  USER="postgres"
  PASSWORD="admin"
  ```
- заполнить файл `.env` значениями (для режима разработки оставить значения `HOST`, `USER`, `PASSWORD` по умолчанию)
- выполнить команду

  ```
  docker compose up
  ```
